# 4. LAYER 3: DECONTEXTUALIZATION

> **Definition:** Source is credible. Quote is accurate. Data is real. But surgically removed from its context — temporal, linguistic, situational, or conceptual. Highly effective because fact-checkers must admit "yes, that was said/published."

## 4.1 Classification Criteria

A piece of information belongs in Layer 3 if ALL of the following are true:

- [ ] The **source is real and verifiable**
- [ ] The **quote/data is accurate** (not fabricated)
- [ ] Critical **context has been removed** that would change the meaning
- [ ] The removal of context was **not accidental** (or the effect is as if intentional)
- [ ] A fact-checker would say "yes, they said that, BUT..."

## 4.2 The Academic Framework: Contextomy

**Contextomy** (formal term) — the informal fallacy of selectively excerpting a passage from its original linguistic, situational, or conceptual environment to distort its intended meaning.

Key properties:
- Creates **straw man arguments** (making opponent's position seem extreme)
- Creates misleading **appeals to authority** (making it seem an expert supports a position they don't)
- In academic publishing: **citation context omission** — citing a source while omitting qualifiers, limitations, or contradictory evidence
- Can create **false consensus** in the scientific literature
- Undermines **reproducibility** (other researchers build on incorrect assumptions)
