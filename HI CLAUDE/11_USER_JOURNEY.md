# 11 — THE USER JOURNEY & ROADMAP (start → end, use cases, personas)

> The canonical map of how a real user moves through the Egyptian Awareness Library — from arrival to
> certified cognitive immunity — plus the "I have a problem right now" use-case index and persona
> fast-paths. Grounded in the live route list + the 2×4 matrix (`10_AUDIT_WORKLIST.md`).
> This is also the blueprint for the user-facing `/master-roadmap` page.

## 1. THE LINEAR JOURNEY (7 stages)

```
 DISCOVER → BASELINE → LEARN (144-day) → VERIFY/DEFEND (tools) → TRACK → CERTIFY → COMMUNITY
   why         measure      inoculate         use as needed        progress    proof      sustain
```

### Stage 1 · DISCOVER — "why does this exist?"
Entry: `/` (landing) → `/philosophy` (the constitution, the One Law) → `/six-layers` (the 8-layer
deception taxonomy, the mental model) → `/platform-guide` · `/why-us` · `/about`.
*Goal:* understand that this is a cognitive **defense system**, not a fact-check site.

### Stage 2 · BASELINE — "how susceptible am I right now?"
`/assessment` (**MIST-20**, validated) → `/baseline` → `/bias-fingerprint` → `/reaction-test`.
*Goal:* a real starting score (pre-inoculation). Re-taken later to measure change (pre/post design).

### Stage 3 · LEARN — the 144-day Cognitive Immunization Curriculum
`/curriculum/phase0` (calibration) → `phase1` (inoculation induction) → `phase2` (epistemic literacy /
Bayesian) → `phase3` (Islamic epistemological defense) → `phase4` (capstone). Supported by the
cognition-builders: `/epistemology`, `/critical-thinking`, `/fallacy-engine`, `/manipulation-cards`,
`/bad-news` (active inoculation), `/cognitive-lab`. Map: `/master-roadmap`.
*Goal:* build durable resistance via graduated prebunking (Inoculation Theory).

### Stage 4 · VERIFY / DEFEND — the toolbox (used on demand, any time)
The 2×4 matrix of tools (§3). Verification engines check a specific artifact; domain shields protect a
vulnerable group. *Goal:* apply the skill to real content the moment it arrives.

### Stage 5 · TRACK — "am I actually improving?"
`/dashboard` (XP, streaks) → `/inoculation-passport` (6 competency domains) → `/comb-tracker`
(COM-B behavior) → `/effect-dashboard` + `/transformation` (pre/post effect sizes) → `/self-test-protocol`
(spaced boosters — decay-resistant). *Goal:* measured, honest progress.

### Stage 6 · CERTIFY — proof of competency
`/certificate` (awareness credential, unlocked at curriculum completion).

### Stage 7 · COMMUNITY — sustain & defend others
`/connect` · `/peer-challenge` (fact-check duels) · `/family-kit` · `/global-alliance` · live intel:
`/threat-map` · `/threat-briefing` · `/trend-hunter` · `/misinfo-atlas` · `/kill-list`. Institutions:
`/supervisor`.

## 4. PERSONA FAST-PATHS

- **University student (18–25 — the core N=84 cohort):** `/assessment` → `/curriculum/phase0…4` →
  `/dashboard` → `/certificate`. Tools on demand.
- **Parent / family guardian:** `/family-kit` → `/whatsapp-analyzer` (for family forwards) →
  `/mental-health` → `/womens-shield` / `/mens-shield`.
- **Religious user:** `/religion-hub` → `hadith-check` / `fatwa-context` / `quran-context` →
  `/curriculum/phase3` (Islamic epistemological defense).
- **Journalist / skeptic / OSINT:** `/deepreal-forensics` → `/forensic-c2pa` → `/reverse` →
  `/paper-auditor` → `/angry-debunkers` → `/evidence`.
- **In crisis (mental health):** `/mental-health/depression` — PHQ-9 + the item-9 safety gate + real
  Egyptian crisis lines, immediately.

gains → get certified → protect your family and community.** Every step is grounded in real sources,
shows its evidence, and abstains when it can't prove a claim (the One Law).
